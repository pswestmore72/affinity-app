## GM Workflow

### Session Prep
1. **Review Basic Rules** - Ensure you understand core mechanics
2. **Build Encounters** - Use Bestiary + scaling guidance from GM Guide
3. **Prepare NPCs** - Use blank template or adapt existing sheets
4. **Set Stakes** - Define success/failure conditions clearly
5. **Plan Flexibility** - Prepare multiple resolution paths

### During the Session
- Keep Quick Reference handy for thresholds and tables
- Use NPC sheets for quick stat lookups
- Note Resolve and HD changes to track narrative tension
- Let players drive story within established boundaries

### After the Session
- Award XP based on encounter difficulty and roleplay
- Distribute loot from Items list
- Note character development and campaign threads
- Plan next session's encounters

## Creating Custom Content

### Custom NPCs
1. Start with [NPC_Sheet_Blank.md](NPC_Sheet_Blank.md)
2. Set role (Minion/Elite/Boss/Legendary)
3. Scale HD based on party level (see GM Guide)
4. Define 2-3 signature abilities
5. Write tactics/behavior section
6. Playtest and adjust

### Custom Monsters
1. Follow Bestiary format
2. Consider creature's role in encounter
3. Balance offensive/defensive capabilities
4. Add unique mechanical twist (resist, vulnerability, special ability)
5. Include roleplay personality notes

### Custom Items
See [Player Book/Affinity_Items.md](../Player%20Book/Affinity_Items.md) for templates and balance guidelines.
