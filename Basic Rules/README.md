# Basic Rules

Core system mechanics and quick reference materials for the Affinity TTRPG.

## Contents

### [Affinity_README.md](Affinity_README.md)
Introduction to the Affinity system, design philosophy, and overview.

### [Affinity_Core_Mechanics.md](Affinity_Core_Mechanics.md)
Fundamental rules for dice rolling, action resolution, contests, and core gameplay loops.

**Key Concepts:**
- d6 dice pools
- Success thresholds (2+, 3+, 4+, 5+, 6+)
- Explosive 6s (each 6 = 2 successes)
- Exertion burn mechanics
- Concentration rules
- Social duels and contests

### [Affinity_Combat_Rules.md](Affinity_Combat_Rules.md)
Complete combat system including initiative, actions, defense, and tactical options.

**Covers:**
- Turn structure
- Attack and defense rolls
- Armor and temporary HD
- Movement and positioning
- Special combat actions

One-page reference for common rules, tables, and mechanics.

**Quick Lookups:**
- Success thresholds table
- Character resources by level
- Hit Dice progression
- Skill list
- Action tiers
