Core Design Philosophy for Affinity Pages

Affinity page design should evoke:

Cosmic order, not medieval fantasy

Precision, not ornamentation

Symmetry, not chaos

Geometry, not vines

Energy and metaphysical structure, not parchment texture

Visual inspirations:

Sacred geometry

Astronomical charts

Technical manuals

Esoteric diagrams

Modern dark sci-fi minimalism

NOT:

Scroll borders

Fake parchment

Calligraphy overload

Decorative clutter

Global Page Framework (Use everywhere)
Margins
Top:    0.75"
Bottom: 0.75"
Inside: 0.9"
Outside:0.75"


Plenty of white space for readability.

Header Style

Left page header:

AFFINITY CORE RULES


Right page header:

Font:

Small caps

9 pt

Light weight

Footer Style

— 42 —


or

Clean. Simple. Intentional.

Template 1: Core Rules Page

Use for mechanics explanations.

Example layout:

[HEADER]

SECTION TITLE

Intro paragraph text explaining concept.

◇ SUBSECTION

Body text explaining mechanics in detail.

◇ RULE

Indented rule block with separator line.

Example text.

◇ DESIGN INTENT

Optional GM explanation or philosophy.

[FOOTER]


Visual Example

EXERTION

All power requires commitment. All commitment carries risk.

◇ COMMITTING EXERTION

When you commit Exertion, you select a number of Hit Dice and roll them.

Each die represents strain placed upon your body and spirit.

◇ RESOLUTION

Each die showing 4+ is a success.

Each die showing 1 causes backlash.

─────────────── ◇ ───────────────

◇ DESIGN INTENT

Exertion ensures power always carries meaningful risk.

Printer Friendly Rules

Avoid:

Background textures

Full page color

Heavy borders

Dark backgrounds

Use:

Thin lines only

Symbols instead of decoration

White background always

This keeps printing cheap.

Ink-efficient design trick

Instead of heavy boxes, use:

──────────── ◇ ────────────


instead of filled shapes.

Affinity-specific identity trick

Use symmetry in layout.

Example:

◇ EXERTION ◇


instead of

EXERTION


This reinforces system theme subconsciously.
