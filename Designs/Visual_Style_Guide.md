## Table Layouts

### Style 1: Clean Minimal (Recommended for Quick Starts)

```
┌─────────────────────────────────────────┐
│ HEADER TEXT                    [accent] │
├─────────────────────────────────────────┤
│ Row 1                            Value  │
│ Row 2 (alternating 5% gray)      Value  │
│ Row 3                            Value  │
└─────────────────────────────────────────┘
```

**Specifications:**
- 1pt borders in muted purple (#5D5878 from web palette)
- Header row: 15% gray OR light accent wash
- Left border accent: 3pt solid gold (#C4A77D) for important tables
- No heavy black borders - keeps it modern

---

### Style 2: Left-Accent Bar (Perfect for Stat Blocks)

```
┃ CREATURE NAME
┃ HD: 4d8  |  Prof: +2  |  Resolve: 4
┃ 
┃ Abilities listed here...
┃ 
```

**Specifications:**
- Heavy 3-4pt left border in gold or purple accent
- No other borders
- Light gray background (#F5F5F5)
- Creates visual weight without clutter

---

### Style 3: Cosmology Tables (For Virtue/Affinity Content)

```
╔═══════════════════════════════════════╗
║   ◆ VIRTUE NAME ◆                     ║
╠═══════════════════════════════════════╣
║   Description text here...            ║
╚═══════════════════════════════════════╝
```

**Specifications:**
- Double-line borders for sacred/important content
- Use sparingly - chapter dividers, key concepts
- Add simple geometric symbol as header icon

## Character & NPC Card Layouts

### Compact Card (1/4 page - fits 4 per page)

```
┌────────────────────────────┐
│ [PORTRAIT    ] NAME        │
│ [SILHOUETTE  ] Class • L1  │
│ [60x60px     ]             │
├────────────────────────────┤
│ HD: 5d10    Resolve: 5     │
│ Prof: +1    Defense: 1     │
├────────────────────────────┤
│ Key Ability: Brief desc    │
│ Equipment: Sword, armor    │
└────────────────────────────┘
```

**Use for:** Quick NPC references, pre-generated characters, monster cards

---

### Two-Column Card (1/2 page - for important NPCs)

```
┌─────────────────────────────────────────────┐
│                                             │
│  [LARGER PORTRAIT]    NAME THE DESCRIPTOR   │
│  [100x120px area ]    "Quote or tagline"    │
│                                             │
├──────────────────────┬──────────────────────┤
│ HD: 6d8              │ Motivation:          │
│ Prof: +2             │ What they want       │
│ Resolve: 6           │                      │
│ Defense: 2           │ Secret:              │
│                      │ What they hide       │
├──────────────────────┴──────────────────────┤
│ Abilities: Key ability name and brief desc  │
│ Equipment: Notable items they carry         │
└─────────────────────────────────────────────┘
```

**Use for:** Named antagonists, quest-givers, recurring characters

---

### Sidebar Card (Margin placement)

```
Perfect for page margins:
- 40-50mm wide
- Thin gold border
- Name + 2-3 key stats
- Small portrait/icon
- "See page X" reference
```

**Use for:** Quick reference callouts, related content links

## Art Style Recommendations

### Primary Style: Ink Line Art with Minimal Shading

**Why this works:**
- Fast to produce
- Reproduces well in B&W printing
- Scans cleanly
- Timeless indie RPG aesthetic
- Matches celestial/ancient tome feel

**Technique:**
```
- Bold outlines (1-2mm for figures)
- Thinner detail lines (0.5mm)
- Cross-hatching for shadows (NOT gradients)
- Leave white space - don't over-render
- Think: old woodcuts meet modern simplicity
```

**Inspiration:** Whitehack, The Black Hack, early D&D modules, Troika!

## Visual Elements to Create

### 1. Chapter Headers / Section Dividers

Simple options:
- Horizontal line with centered geometric symbol `——◆——`
- Repeating small icon (stars, moons, crystals)
- Thin decorative bracket frames

**Production:** Create 3-4 reusable dividers in Affinity Designer. Export as PNG with transparency.

---

### 2. Class Icons (7 total)

Simple silhouette or symbol for each class:

| Class | Icon Concept |
|-------|-------------|
| **Combatant** | Crossed swords or shield |
| **Priest** | Radiant sun/holy symbol |
| **Ranger** | Arrow or animal track |
| **Scholar** | Open book or eye |
| **Sovereign** | Crown or single flame |
| **Weaver** | Spiral or geometric pattern |
| **Shaman** | Spirit mask or antlers |

**Style:** 1-2 colors max, geometric, no fine detail  
**Size:** 15-20mm square, scalable vector

---

### 3. Spot Illustrations (Small)

Scattered throughout text:
- Dice illustrations (for HD system!)
- Weapon silhouettes
- Simple hands holding items
- Geometric "affinity" symbols
- Small creature silhouettes

**Size:** 30-50mm wide  
**Placement:** Text wrap around, or margin placement

---

### 4. Half-Page Illustrations (1-2 per guide)

For major section breaks:
- A single adventurer silhouette
- Dramatic scene (only 1-2 figures)
- Environmental mood piece (ruins, stars, forest)

**Keep it simple:**
- No complex backgrounds
- 1-2 values (black + one gray wash)
- Strong compositional shapes

### Color Usage in Print

**For playtest PDFs (screen viewing):**
- Use web accent colors sparingly
- Gold (#C4A77D) for headers, accents
- Purple (#9B8BB8) for highlights
- Keep art predominantly BLACK

**For future print:**
- Design in grayscale first
- Add spot color later as overlay
- Everything should work without color

## Visual Hierarchy

```
Page scan order (where eyes go):
1. Chapter header (largest)
2. Illustration (if present)
3. Section headers
4. Tables
5. Body text
6. Margin notes/icons
```

**Visual weight distribution:**
- 60% white space + body text
- 25% structural (headers, tables, rules)
- 15% illustration/decoration

## Inspiration Games

**Indie RPG Art Worth Studying:**

| Game | What to Learn |
|------|---------------|
| **Mörk Borg** | Bold contrast, chaotic energy (probably TOO extreme for Affinity) |
| **Troika!** | Whimsical line art, great use of white space |
| **The Black Hack** | Minimal, functional, clean |
| **Whitehack** | Academic simplicity, text-focused |
| **Cairn** | Modern minimal, excellent tables |
| **Electric Bastionland** | Character art style, economical lines |
| **Heart: The City Beneath** | Mood through silhouette |
| **Blades in the Dark** | Elegant darkness, spot illustrations |

**Target aesthetic:** Troika! simplicity + Blades in the Dark elegance + Affinity celestial color accents
