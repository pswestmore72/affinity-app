# Player Book

Everything players need to create and play characters in the Affinity TTRPG.

## Contents

### Character Creation

#### [Affinity_Character_Creation.md](Affinity_Character_Creation.md)
Step-by-step guide to building your character from Level 1 to Level 10.

**Includes:**
- Starting resources by level
- Skill selection
- Affinity (virtue) system
- Level progression tables

### Classes

#### [Affinity_Classes.md](Affinity_Classes.md)
Complete class guide with all 7 playable classes, abilities by level, and design notes.

**The Seven Classes:**
- **Combatant (d10):** Oath-Bound Warrior, Battlefield Master
- **Priest (d8):** Invoker, Resolve Anchor, Space Sanctifier
- **Ranger (d8):** Wilderness Master, Terrain Controller
- **Scholar (d8):** Knowledge Weaponizer, Pattern Identifier
- **Weaver (d6):** Reality Bender, Spell Supremacist
- **Shaman (d6):** Spirit Speaker, Threshold Walker
- **Sovereign (d10):** Self-Sufficient Master, Balance Keeper

## Reading Order for New Players

1. **Character Creation:** Start with [Affinity_Character_Creation.md](Affinity_Character_Creation.md)
2. **Choose Your Class:** Read relevant section in [Affinity_Classes.md](Affinity_Classes.md)
3. **Magic (if applicable):** If your class casts spells, read [Affinity_Magic_System.md](Affinity_Magic_System.md)
4. **Path Progression:** Understand [Affinity_Path_Board.md](Affinity_Path_Board.md) for customization
5. **Equip Yourself:** Browse [Affinity_Items.md](Affinity_Items.md) for gear

Don't forget to review the [Basic Rules](../Basic%20Rules/) for core mechanics!
